Problem solving

During supporting the project I achieved our company client satisfaction using my communication and problem solving skills...

  • While working as a consultant in Saudi Arabia I was employed on a project in the Saudi post office, who had severe performance issues with the core payroll services running on their Oracle platform. I was brought in to help to optimize the payroll program and to facilitate a more streamline process within the Post Office HR and Payroll Dept.
  • Initial investigations centered around generating the data to troubleshoot the problem. Using the Oracle tracing program, I generated a diagnostic trace file which facilitated analysis of the issue. My analysis revealed that a lot of the current table scans were redundant causing the performance issue. To solve this, I created indices and re-wrote the SQL query reducing the amount of redundant table scans incurred. Once I had performed these fixes I ran testing on a clone test environment and the key stakeholder signed off the changes, I deployed it in the live environment.
  • Through production testing the changes had optimized the payroll process from 6 hours to 10 minutes, satisfying the initial requirement that I had been assigned.
